# PMO Suite 项目交接文档

## 📦 项目信息

| 项目 | 值 |
|------|-----|
| 名称 | PMO Suite（游戏行业项目管理） |
| 技术栈 | React 19 + Vite 8 + Tailwind CSS 4 + Recharts + Lucide |
| GitHub | `https://github.com/1751575290/PMO` |
| 分支 | `master` |

---

## 🚀 在新电脑上启动（复制这段直接粘贴给 AI）

```
git clone https://github.com/1751575290/PMO.git
cd PMO
npm install
npx vite --host 0.0.0.0 --port 5200
```

## 📋 项目中已完成的系统提示词

让新电脑上的 AI 继续工作，用这段提示词：

> 请帮我继续开发 PMO Suite 项目管理前端项目。项目源码在刚才 clone 的 PMO 目录中。
>
> **已实现的功能：**
> - 14 个页面：Dashboard、ProjectList/Detail/Create/New、GanttChart、ResourceManagement、ApprovalCenter、RiskManagement、DataAnalytics、Settings、Login、GRReview（三 Tab：材料审查/AI解析/评分结果）
> - 暗夜/白昼双主题切换（Settings 页顶部开关，ThemeContext + CSS 变量方案）
> - Dashboard 全卡片下钻（统计卡片→筛选列表、饼图→状态筛选、柱状图→部门弹窗、趋势图→月度明细、最近项目→详情页）
> - ProjectList 支持 URL 参数自动筛选（?status= & ?dept= & ?overdue=1）
> - 游戏行业 8 个代号项目：星辰(MMO)、破晓(战术竞技)、幻塔(卡牌)、龙魂(ARPG)、暗潮(FPS)、山海(休闲)、极光(三消)、永恒(SLG)
> - 甘特图分在研/在营双轨时间线
> - 路由：HashRouter，base: './'
>
> **主题文件夹：**
> - 源码在 PMO/ 根目录
> - 主题控制：src/context/ThemeContext.jsx
> - 主题样式：src/index.css（CSS 变量 :root + .theme-dark）
> - Tailwind 设计令牌：tailwindcss @theme 配置
>
> **已知注意：**
> - React 19 + Vite 8，不要降级
> - Tailwind v4，使用 @tailwindcss/vite 插件
> - JSX 中 > 写成 {'>'}
> - Lucide-react 图标库版本 1.16+
>
> 请先 npm install && npx vite 启动项目，然后在浏览器中预览确认正常。之后请根据我的新需求继续开发。
> 每次改动后请帮我自动 git commit && git push。

## 🗂 项目文件结构

```
PMO/
├── index.html
├── package.json
├── vite.config.js       # base: './'
├── public/
│   ├── favicon.svg
│   └── icons.svg
└── src/
    ├── main.jsx
    ├── App.jsx          # 路由注册
    ├── index.css        # 全局样式 + CSS 变量 + Tailwind
    ├── components/
    │   ├── Layout.jsx   # 壳（侧边栏 + 页头 + Outlet）
    │   └── Sidebar.jsx  # 8 个导航项 + 用户卡片
    ├── context/
    │   ├── ThemeContext.jsx    # 暗夜/白昼主题
    │   └── ProjectContext.jsx  # 项目状态管理
    ├── data/
    │   └── projects.js         # 初始数据 + 工具函数
    └── pages/
        ├── Dashboard.jsx       # ⭐ 总览（下钻已完成）
        ├── ProjectList.jsx     # ⭐ 项目列表（URL筛选已完成）
        ├── ProjectDetail.jsx
        ├── ProjectCreate.jsx
        ├── NewProject.jsx
        ├── GanttChart.jsx      # 项目规划（在研/在营双轨）
        ├── ResourceManagement.jsx
        ├── ApprovalCenter.jsx
        ├── RiskManagement.jsx
        ├── DataAnalytics.jsx
        ├── Settings.jsx        # ⭐ 系统设置（含主题切换）
        ├── Login.jsx
        ├── GRReview.jsx        # GR 评审（三 Tab，大文件）
        └── GRAIScoring.jsx     # 已废弃
```
